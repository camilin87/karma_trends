# Karma Trends
Our hack summit hackathon project

## Deployment

### Dependencies
    sudo apt-get install -y node
    sudo apt-get install -y nodejs-legacy
    sudo apt-get install -y npm

### Execution
    cd <repo_path>
    git pull && npm run deploy

## Development

### Dependencies
    npm install http-server -g
### Execution
    http-server
